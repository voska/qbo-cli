# Sandbox and Testing Tools

There are multiple ways to test your app during development.

You can start with the sandbox QuickBooks Online company connected to your Intuit Developer account. This gives you sample data to test with.

- [Create and test with a sandbox company](https://developer.intuit.com/app/developer/qbo/docs/develop/sandboxes/manage-your-sandboxes)

You can also set up and configure third-party apps, like Postman, to work with our API frameworks.

- [Test with Postman](https://developer.intuit.com/app/developer/qbo/docs/develop/sandboxes/postman)
